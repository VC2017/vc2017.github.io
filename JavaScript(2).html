JavaScript 运算符

  1. 比较运算符
    var x=5；
运算符 描述 例子
== 等于 x==8返回false
=== 全等（值和类型） x===5返回true；x===”5”返回false
!= 不等于 x!=5返回false
> 大于 x>8返回false
< 小于 x<8返回true
.>= 大于或等于 x>=8返回false
<= 小于或等于 x<=8返回true
  1. 如何使用
    使用条件语句（if/while）进行判断根据返回结果进行使用
  2. 条件运算符
    var x=3；
    var y=4；
运算符 描述 例子
&& and（且） （x>3 && y<6）返回false
或 or（或） (x>3
！ not（非） !(x==y)返回true
  1. 条件运算符
    和C++中三目运算符相同

javaScript if….else语句

js中if….else语句和C++基本相同

if(条件){
    document.write("若条件成立则执行");
}else{
    document.write("若条件不成立则执行");
}

if….else if….else

if(条件一){
    document.write("若条件一成立则执行");
}else if(条件二){
    document.write("若条件二成立则执行");
}else{
    document.write("若条件一，条件二都不成立则执行");
}

javaScript switch（）

switch选择执行代码

var n=...
switch(n)
   {
   case 1:
     执行代码块 1
     break
   case 2:
     执行代码块 2
     break
   default:
     如果n即不是1也不是2，则执行此代码
   }
   //switch(可以为变量也可以为表达式)
