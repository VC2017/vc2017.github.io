JavaScript

1 什么是JavaScript

  • JavaScript被设计用来向HTML页面添加交互行为
  • JavaScript是一种轻量级的编程语言
  • JavaScript通常被直接嵌入HTML页面
  • JavaScript是一种解释性语言

JavaScript的实现

  • 如果需要把一段 JavaScript 插入 HTML 页面，我们需要使用 
<html>
<head>
<title>实例</title>
</head>
<body>
<script type="text/javascript">
document.write("Hello Word!")
</script>
</body>
</html>

上述代码会在HTML页面中显示：Hello Word！
document.wirte:用于向页面输入

  1. 外部实例
<script src="#"></script>
    //新建一个javaScript文件 由src引入(#为文件放置地址)

JavaScript的放置

  • JavaScript可以放置在任意位置区别在于加载JavaScript与body、head等的先后

JavaScript语句

<script>
document.write("<h1>This is a h1</h1>");
</script>
  • JavaScript语句通常最后要加;浏览器按照顺序依次执行每条语句

JavaScript注释

  • 注释分为单行注释和多行注释和C语言相同
  • 单行注释：//
  • 多行注释
    注释：学会更好地使用注释在更改代码和浏览代码时起到更好的帮助

JavaScript变量

  • JavaScript变量可以保存数字，字母也可以保存文本值
  • 变量对大小写敏感(y,Y)是两个不同的变量
  • 变量名必须以字母或下划线开始
  • 定义变量用 var
    实例：
var x=5;
var name="zhang"

在JavaScript中变量未声明时会自动声明如：x=5；

JavaScript运算符

var y=7;

运算符 描述 例子 结果
+ 加 x=y+2 x=9
- 减 x=y-2 5
* 乘 x=y*2 14
/ 除 x=y/7 1
